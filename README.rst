Install
------------------------------------------------------------------------------

``learn_sphinx`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install learn_sphinx

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn_sphinx
